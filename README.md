# Do or Die

Do or Die is a web-based multiplayer board game (work in progress) where players can dare each other to do actions in order to progress along the board. Players can join game rooms hosted by other users via generated room codes. This repository contains ui code in order to run the game

# How To Start

Run api located: https://github.com/kt-kenny/DoOrDieBoardGame
`npm i`
`npm run dev`

Currently developed with Node v22.17.1
